<objective>
Build the schema symbol index and LanguageClient middleware that scopes completions, hover, and signature help per-file based on load() imports.

Purpose: The LSP provides all schema symbols globally via --builtin-paths. This plan adds the filtering layer: only symbols explicitly imported via load() in the current file (plus builtins) appear in IntelliSense. This is the core "per-file scoping" feature.

Output: Two modules (`src/schema-index.ts`, `src/middleware.ts`) with comprehensive unit tests. No extension integration yet -- Plan 04 wires these into the LanguageClient.
</objective>

  <action>
    Create `src/middleware.ts` exporting:

    - `getAllowedSymbols(documentUri: string, documentText: string, schemaIndex: SchemaIndex): Set<string>` -- calls `parseLoadStatements(documentText)` to get load statements. For each statement, if symbols includes "*", add all symbols from `schemaIndex.getSymbolsForFile()` using the appropriate relative path. Otherwise, add each named symbol. Always include BUILTIN_NAMES. Returns the union set.

    - `createScopingMiddleware(schemaIndex: SchemaIndex, getDocumentText: (uri: string) => string | undefined)` -- returns an object with middleware hooks:

      ```typescript
      {
        provideCompletionItem: async (document, position, context, token, next) => {
          const result = await next(document, position, context, token);
          if (!result) return result;
          const text = getDocumentText(document.uri.toString()) ?? document.getText();
          const allowed = getAllowedSymbols(document.uri.toString(), text, schemaIndex);
          // Handle both CompletionItem[] and CompletionList (Pitfall 6)
          const items = Array.isArray(result) ? result : result.items;
          const filtered = items.filter(item => {
            const label = typeof item.label === 'string' ? item.label : item.label.label;
            return allowed.has(label);
          });
          return Array.isArray(result) ? filtered : { ...result, items: filtered };
        },
        provideHover: async (document, position, token, next) => {
          const hover = await next(document, position, token);
          if (!hover) return hover;
          const wordRange = document.getWordRangeAtPosition(position);
          if (!wordRange) return hover;
          const word = document.getText(wordRange);
          const text = getDocumentText(document.uri.toString()) ?? document.getText();
          const allowed = getAllowedSymbols(document.uri.toString(), text, schemaIndex);
          return allowed.has(word) ? hover : undefined;
        },
        provideSignatureHelp: async (document, position, context, token, next) => {
          // Pass through -- signature help triggers only inside function calls,
          // and the function name is already scope-checked by completions
          return next(document, position, context, token);
        },
      }
      ```

    - Internal: `documentImportsCache: Map<string, Set<string>>` -- caches allowed symbols per document URI. `updateDocumentImports(uri: string, text: string, schemaIndex: SchemaIndex): void` -- re-parses and updates cache. `clearDocumentImports(uri: string): void` -- removes from cache (for closed documents).
